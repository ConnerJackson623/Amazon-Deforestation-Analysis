# Amazon Deforestation Analysis

## Overview

This project uses ArcGIS Pro to analyze deforestation in Rondônia, Brazil. The analysis combines satellite imagery, transportation networks, protected forest boundaries, and land cover data to identify areas experiencing forest loss and evaluate how infrastructure influences environmental change.

---

## Skills Demonstrated

- Raster Data Analysis
- Vector Data Integration
- Spatial Analysis
- Environmental GIS
- Protected Area Analysis
- Transportation Network Analysis
- Cartographic Design
- Map Layout Creation

---

## Workflow

1. Load raster and vector datasets into ArcGIS Pro.
2. Display satellite imagery and land cover information.
3. Overlay protected forest boundaries.
4. Add road networks and transportation infrastructure.
5. Identify areas of active deforestation.
6. Create a professional map layout.
7. Export the final analysis as a PDF.

---

## Results

The completed map highlights regions where deforestation overlaps with transportation corridors and protected forests. This type of analysis supports environmental monitoring, conservation planning, and land management decisions.
